### 阅读模型
>能力层次（德雷福斯模型）
 
| 阶段 | 特征 | 总结 |  
| :-: | - | - |  
| 新手 | 新手需要指令清单| 一无所知 |  
| 高级新手 | 高级新手不想要全局思维 | 知道（实用技术） |  
| 胜任者 | 胜任者能够解决问题 | 理解（内部原理） |  
| 精通者 | 精通者能够自我纠正 | 道（本质规则） |  
| 专家 | 专家评直觉工作 | 道法自然 |
